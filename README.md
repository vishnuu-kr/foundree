# Foundree.dev

We partner with founders, creators, and dreamers who want to build cool shit without the corporate red tape. Let's open a repo, turn on some music, and make it real.

## Tech Stack

This project is built with:
- **Vite**
- **TypeScript**
- **React**
- **Tailwind CSS**
- **shadcn/ui**
- **Framer Motion**

## Running Locally

Clone the repository and install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Features
- **Interactive UI**: Magnetic buttons, scramble text, and smooth scrolling.
- **Terminal Comms Protocol**: Fun easter egg (`⌘K`) for the interactive contact terminal.
- **Vibe Background**: Custom shaders and visual effects natively built in React.

## Deployment

Deploy this project on Vercel, Netlify, or any static hosting provider.

```bash
npm run build
```
